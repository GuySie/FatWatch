Release Notes

1.4 - Submitted January 14, 2009; Ready for Sale January 16, 2009

Body Mass Index (BMI) Support: enable on the More tab. After selecting your height, BMI will be indicated in other views by color: red for obese, yellow for overweight, green for normal, blue for underweight.

Log:
	- Go To: larger buttons.
	- Improved drawing of long notes.
	- Segmented control to toggle between display of variance (distance from the moving average line) or BMI (if enabled).
	- Added a "hint" table cell in the first section.

Weigh-in:
	- Changed background color.
	- If BMI is enabled, weight picker is colored by BMI zone.
	- Note field now supports multiple lines and has sexy rounded corners.

Trends:
	- Added overeating, undereating labels to energy trend.
	- Show total weight gained/lost over the given period.

Goal:
	- Goal weight is set using whole numbers, regardless of scale precision setting.
	- If BMI is enabled, see start BMI, set a goal BMI.
	- Added info button to explain how start weight is computed.

More:
	- Added switch to enable BMI monitoring.
	- Added a button to review FatWatch in the App Store.
	- Removed unnecesary section titles, added hint about Settings app.

Chart:
	- Linked scrolling to Log: Browse mode starting position is determined by current Log position.
	- Regions are shaded based on BMI (if enabled).
	- If database is empty, displays an empty chart.
	- Offers several preset zoom levels: Past Month, Past Quarter, Past Year, All Time, and Browse (the original scrollable view).
	- Labels are drawn using Unicode, for better international support.

General:
	- Added a startup image.
	- When you quit FatWatch, it remembers which tab you had selected (i.e., Log, Trends, Goal, or More) and returns to it when you next launch the app.
	- Setup screen respects existing preferences.
	- Added a Cancel button to the Set Passcode screen.
	- Wi-Fi Import/Export displays device name (e.g., "iPhone" or "iPod touch") as appropriate.
	- Fixed several crashing bugs.

Settings:
	- Renamed "Scale Increment" to "Scale Precision".
	- Added a switch to disable Bonjour (if you don't want to publish your address when you turn on Wi-Fi Import/Export).
	- Shade Weekends now defaults to on.
	- Added a Shade BMI Zones to control BMI shading in chart (default: on).

1.3 - Submitted September 10, 2008
	
- Chart: graph segments are drawn in a secondary thread for smoother scrolling
- Fixed an issue where the chart would not draw if too little data was available.
- Fixed an issue where trend calculation would ignore the last day of the month in some cases.
- Chart: replaced circules with rhombuses, highlight color is now blue (same as check mark)
- Trends: includes a "Since Goal Start" when enough information is available to calculate

1.2 - not released to public
